This dashboard provides a comprehensive and interactive overview of the Paris 2024 Olympic Games. It is designed to visualize key data such as the number of medals, participating countries, athlete statistics, and event schedules.

Users can easily explore performance insights through dynamic charts and filters, allowing them to compare countries, track medal rankings, and analyze trends across different sports. The interface focuses on clarity and usability, making complex data easy to understand at a glance.

I used Power Query to import and transform data, and DAX to create measures for dynamic and interactive visualizations.
